# CRM-Demo

This repository contains the CRM Demo, structured as a clean monorepo containing both the backend and frontend components.

## Project Structure

- **`/backend`**: The C# backend solution (`CRM.slnx`) and projects (ASP.NET Core Web API, Business logic layer, DataProvider, etc.).
- **`/frontend`**: Reserved directory for the Angular/Frontend web application.

## Getting Started

### Backend
To get started with the backend, navigate to the `backend/` folder and open the `CRM.slnx` solution file in Visual Studio 2022 or higher, or run the following command using the .NET CLI:

```bash
cd backend
dotnet build
```
